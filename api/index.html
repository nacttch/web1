API Reference

POST /v1/game/init

Initializes a new game session for a specific player. This endpoint returns the initial RNG seed and balance configuration.

Request Example

curl -X POST https://api.gamevy.com/v1/game/init \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "game_id": "lucky_slots_01",
    "currency": "GBP"
  }'

Parameters

game_id string (required)
currency string (optional)
